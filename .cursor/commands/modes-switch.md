# Modes switcher

Use the following prompts as an input when you're asked inside another command:

- **Ask**: Switch to **Ask mode** before proceeding. Do not make any edits to code
- **Plan**: Switch to **Plan mode** before proceeding. Do not make any edits to code
- **Agent**: Switch to **Agent mode**. Execute what has been asked
- **Debug**: Switch to **Debug mode**. Change code only in bug fixing context
